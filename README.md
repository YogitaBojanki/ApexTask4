# 🍔 Food Ordering System (Task 4)

## 📌 Project Overview

This project is a **Food Ordering System** developed using **HTML, CSS, Bootstrap, PHP, and MySQL**.
It allows users to register, login, browse food items, add items to cart, and place orders with a smooth and responsive user interface.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* PHP
* MySQL
* XAMPP

---

## ✨ Features

* User Registration & Login Authentication
* Session-based Access Control
* Responsive Food Menu UI
* Add to Cart Functionality
* Dynamic Cart Management (+ / - / Remove)
* Order Placement System
* Order Success Page with Payment Options
* Logout Functionality
* Login Restriction (Access control for menu & cart)

---

## 🔐 Security Features

* Session-based authentication
* Restricted access to menu & order pages
* Input validation

---

## 📊 Functional Flow

1. User registers and logs in
2. User accesses food menu
3. Adds items to cart
4. Views cart and updates quantity
5. Places order
6. Receives order confirmation

1. Install XAMPP
2. Start Apache & MySQL
3. Place project inside `htdocs`
4. Import database in phpMyAdmin
5. Open browser:
   http://localhost/ApexTask4/
